Find an Article 4 Direction

Last updated: 18 March 2026

Search and filter to find Article 4 Directions in West Brackenfield District. An Article 4 Direction removes certain permitted development rights, meaning planning permission must be applied for before carrying out the affected works.

About this example: Filters and pagination are reflected in the page URL as query parameters (for example, finder.html?q=conservation&type=external-alterations&page=2). This means filtered views and individual pages of results can be bookmarked and shared. In a real implementation a server would use these parameters to query a database and return the appropriate records.
